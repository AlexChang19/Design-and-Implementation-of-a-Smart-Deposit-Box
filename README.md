# 智能置物箱之設計與實作 (Design and Implementation of a Smart Deposit Box)

## 研究背景

### NFC手機市占率
關於NFC技術的必要性，在2012年Google Android 手機上開始全面支援NFC功能，2015年全球近場通信（NFC）市場規模估值為48億美元。

## 雛型設計
使用手機感應置物櫃上的 NFC 感應區，便可觸動電磁閥做動，從而使置物櫃門自動打開。
