# 智能置物箱之設計與實作 (Design and Implementation of a Smart Deposit Box)

# 構思

# 雛型設計
使用者使用手機登入完成之後，感應置物櫃上的NFC感應區，便可觸動一系列動作，使電磁閥做動，從而使置物櫃門自動打開。
